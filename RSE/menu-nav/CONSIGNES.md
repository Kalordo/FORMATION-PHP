# Rendre le header accessible

## Régler les problèmes d'accessibilité

Dans cet exercice, la nav qui apparait présente plusieurs problèmes d'accessibilité : 

- un problème de contraste
- un problème d'accessibilité au clavier

Faites les modifications nécéssaires pour régler ces deux problèmes.

Notez dans un fichier les modifications que vous avez apportées.

## Bonus

Afin d'éviter à un utilisateur de devoir reparcourir toute la navigation, il y a une amélioration que l'on pourrait faire : ajouter un lien au tout début du site pour accéder directement au `<main>`.

Mettez en place cet ajout pour aider les utilisateurs et utilisatrices.

Si vous souhaitez masquer le lien vous pouvez utiliser la classe CSS `visually-hidden-focusable`.
